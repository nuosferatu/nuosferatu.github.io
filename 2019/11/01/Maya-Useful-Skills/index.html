Maya 实用操作指令

发表于 2019-11-01 分类于 其它
本文字数： 1.2k

同为 Autodesk 的三维软件，Maya 虽然主业是动画、渲染，但其实建模能力也是不容小觑的，比 3ds Max 有着更为方便、精准的操作，只不过较复杂，本文是 Maya 建模中的一些实用指令和操作。

挤出

选中一条边，Ctrl+E 打开挤出工具，调整 xyz 三轴，挤出一条边，实际上是扩展出了一个以这条边为起点的面，如下图：

挤出面，则是拉伸这个面，达到扩展体的效果：

多选

双击和 Shift 双击，可以快速多选。

多切割

多切割工具，可以给多边形添加边，连接点必须在点或边上：

在侧视图、顶视图这种透视图上切割，可以贯穿模型：

复制

Ctrl+D 可以实现立即将选中对象复制一份，复制出来的对象，经过一系列旋转、缩放、移动后，Shift+D 可以实现复制并且做同样的变换，十分方便：

Ctrl+Shift+D 可以做特殊复制，特殊复制的细节可以在 编辑 -> 特殊复制 中修改，可以选择复制一份对象，或者一个实例（即产生的新对象是原对象的实例，如果原对象发生变化，新对象也发生相同的变化），如下图，可以看到，两个 pCube 实际上是同一个形状，只是具有不同的局部坐标系：

对该形状做一些操作（不包含缩放、旋转、移动等变换），两个 pCube 会同时变化：

边缘对齐

修改模型中心轴：点击 D 进入模型中心轴编辑模式，点击一条边、面或者点，将模型的中心轴修改到模型边缘（放心，之后还可以重置中心轴），再次点击 D 退出模型中心轴编辑模式。

按住 V 拖拽模型，此时处于点贴合模式，光标移动到某一个点附件，模型就会跟着移动并对齐至这个点，由于中心轴修改到了边缘上，所以实现了边缘对齐。

倒角

选中边，点击倒角工具：

分数是倒角占整条边的百分比，分数越小，倒角尺寸越小，也就是图中半径越小。

分段数越大，倒角处网格线越多，相应的倒角也越平滑。

深度是倒角的弧面位置，从 -1 到 1，深度为 1 时，圆弧朝外且圆弧对应的圆心角最大，上图是直角，深度为 1 时，圆心角刚好 90 度，1/4 圆弧。深度越小，圆心角越大，小于 0 时，圆弧朝里。

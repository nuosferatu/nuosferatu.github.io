VMware 虚拟机黑苹果 macOS 10.15.4 安装教程

因为想简单地了解一下的 iOS 开发，需要用到 Xcode，自己又没有一个 MacBook，所以只好用虚拟机安装 macOS 试一试咯。

准备工作

首先准备一些必要文件：

  • 系统映像：macOS 10.15 Catalina 或更高版本（cdr 格式）
  • 虚拟机：VMware Workstation Pro，版本 15.5 或以上
  • unlocker（建议下载魔改版 MK-unlocker，以保证对 macOS 10.15 的支持）
  • Chameleon Wizard（用来模拟真实设备信息，以登录 Apple ID 和 App Store）

安装 VMware Workstation

安装 VMware Workstation

安装 VMware Workstation Pro，建议版本 15.5 或以上，否则可能不支持 macOS 10.15 及以上版本。安装过程很简单，不再赘述，激活密钥自行百度。

启用虚拟化

VMware 安装结束后，需要重启电脑。（建议关机前先去 控制面板 -> 卸载程序 -> 启用或关闭 Windows 功能 中取消勾选 Hyper-V，关机后，进 BIOS 启用 虚拟化 功能）

解锁 macOS 选项

开机后，去服务中，关闭 VMware 开头的 5 个服务：

把 MK-unlocker 解压出来，以管理员身份运行 win-install.cmd，运行结束后重新启动刚刚关闭的 5 个服务。（或者干脆 重启电脑，简单粗暴）

新建虚拟机

打开 VMware Workstation Pro，新建虚拟机，选择 典型，系统映像文件选择 cdr 格式的 macOS 10.15 Catalina，虚拟磁盘位置随意，大小建议 60 GB 以上，否则无法安装 Xcode，会提示硬盘空间不足。

安装系统

打开虚拟磁盘目录下，后缀为 vmx 的文件，用文本编辑器打开，找到 smc.present = “TRUE” 这一行，在下面加一行 smc.version = “0”，保存。

运行虚拟机，开始安装系统。安装时，先 抹掉虚拟磁盘，然后执行 安装 macOS，过程相当缓慢，建议泡桶面，打开 B 站，看个视频。

安装 VMware Tools（为了全屏等功能）

选择 VMware Workstation 的 虚拟机 -> 安装 VMware Tools…，这时候，虚拟机会自动挂载 VMware Tools 的安装盘。

别急着安装，先去 系统偏好设置 -> 安全性和隐私 中，允许 VMware Tools 安装扩展，然后再去安装 VMware Tools，否则第一遍可能会安装失败，因为权限不足被拒绝了。安装好后，重启虚拟机即可。

模拟设备信息（为了登录 Apple ID 和 App Store）

将 Chameleon Wizard 解压到虚拟机中，建议不要放在桌面，解压后，运行解压出来的程序，选择 SMBios 标签，左边下拉列表选择一个机型，我选择的 MacBook Pro (8, 1)，然后先拍张照，后面有用，另存为。

重启虚拟机，打开 关于本机，可以看到已经有一个真实的设备信息了，接下来就可以登录 Apple ID 和 App Store 了。

另外，我安装的时候，Xcode 已经需要系统版本至少 macOS 10.15.4 了，所以，如果你安装的系统版本较低，请系统升级，虚拟机里面是可以直接系统升级的，当然，过程更加的漫长，或者一开始就找一个 cdr 格式的 macOS 10.15.4 的系统映像，反正我没找到，所以就安装了 macOS 10.15 后升级到最新版 10.15.4，可以正常安装 Xcode，不过很卡很卡，毕竟是笔记本，如果是台式机，奢侈一点多给一些资源，应该会好很多，至少学学 Swift，iOS 开发入个门还是可以的。

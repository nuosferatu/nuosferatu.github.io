昨天搭建了自己的个人博客，想记录一些学习笔记，分享些实用和有趣的内容。因为自己比较小白，参考了好多文章，踩了无数的坑，终于搭建好了，仍旧存在一些问题，以后慢慢改善。第一篇博客呢，就写一下如何从零开始搭建一个个人博客，包含了我遇到的各种坑，希望可以帮到向我一样的小白。

设置 SSH 和 Git

提示选择密钥生成目录、输入密码和重输密码时，一路回车默认下去，这样在当前的用户目录下就生成了密钥。

打开当前用户目录，即 C:\Users\某用户，可以看到一个文件夹 .ssh，该目录下至少有两个文件，id_rsa 和 id_rsa.pub，用记事本（或者其它文本编辑器）打开文件 id_rsa.pub，复制全部内容。

点击自己的 GitHub 右上角的头像，打开 Setting（设置），在 SSH and GPG keys 标签下找到 SSH keys，点击按钮 New SSH key 添加新密钥，把刚刚复制的内容粘贴到内容域（Key），名称（Title）随便写一个，点击 Add SSH key 完成密钥的添加。

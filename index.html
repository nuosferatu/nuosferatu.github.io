C++ - 智能指针

发表于 2020-05-26 分类于 学习笔记 > C++
本文字数： 2.6k

每个程序有一个内存池，叫做堆（heap），用来动态分配。

  • shared_ptr，允许多个指针指向同一个内存；
  • unique_ptr，独占所指向的对象；
  • weak_ptr，伴随类，弱引用，指向 shared_ptr 指向的对象。

都定义在 memory 头文件中。

异常处理机制组成部分：

  • 异常检查部分：throw

  • 一个异常类，用于 throw 和 catch 传递异常现象。

配置 VS Code

安装 Remote Development Extension Pack，点击这个位置：

选择 Remote-SSH: Connect to Host…，输入 用户名@IP 如 root@192.168.0.137，提示选服务端类型，比如我的是 Linux，然后输密码，回车，就完成连接了。

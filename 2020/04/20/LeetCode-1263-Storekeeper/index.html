LeetCode 1263 题 推箱子。

代码中 // 是算法的注释，// PS: 是笔记。

大致思路如下：

  1. 从地图中找到人和箱子的位置，把地图恢复成单纯干净的地图；
  2. 定义状态：人的位置 + 箱子的位置，还要有一个 set 来保证搜索空间不重复，已经出现过的状态不在添加；
  3. 初始状态加入队列，然后每次从队列中弹出一个状态，从当前状态分别检测向上、下、左、右四个方向移动，是否会产生新状态，产生的新状态加入队列；直到箱子到达终点，或者队列为空；
  4. 由于人的移动不计步数，只有箱子移动计步数，所以队列应当是一个优先级队列，箱子移动步数越少的状态，优先级越高，应该越早被弹出队列，新状态中箱子位置如果发生变动，则步数 +1；
  5. 搜索人可以移动的状态，这个过程按照 BFS 搜索，如果人碰到了箱子，则箱子被推动，移动过程（即新状态）如果不合法，则这一状态不算数，不加入队列；不合法状态包括人撞墙、箱子撞墙、越界；

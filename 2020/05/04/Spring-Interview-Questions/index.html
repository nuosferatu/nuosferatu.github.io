2020 春招实习生笔面试问题整理

发表于 2020-05-04 分类于 面试
本文字数： 5.1k

整理了春招笔试和面试中遇到的一些问题，分类整理汇总了一下，以备秋招准备使用。

编程语言

C++ 语言知识

  1. static 的作用

  2. 讲讲 const

  3. class 和 struct 的区别？在初始化的方法上，有什么区别？

  4. struct 字节对齐，如果 struct 里面套 struct，外层字节对齐是怎样的？

  5. 【堆栈】说说对 C++ 中堆和栈的理解。哪些东西放在堆里面，哪些放在栈里面？

  6. 【this】有哪些显式引用 this 指针的情况？有哪些隐含 this 指针的函数？

  7. 【类继承】讲讲不同权限的继承。继承中函数、变量的问题。C++ 多态的实现方式。子类多态性实现的时期。讲讲虚函数表。讲讲多继承中子类的虚函数表。讲讲钻石继承。

  8. 【虚函数】讲讲虚函数。如何知道调用的函数是哪一个？析构函数通常设计为虚函数，为什么？如果不设计成虚函数，析构过程会发生什么？讲讲动态绑定、虚函数覆盖。

  9. 讲讲对象的内存模型。一个对象如何确定它的成员函数的偏移量？

  10. 讲讲多层 throw-catch，讲讲 catch 类型匹配。

  11. 什么情况会发生内存泄漏？大型项目中如何避免、检查内存泄漏？有什么方法可以避免申请的内存忘记释放？智能指针。

  12. 讲讲智能指针。如何实现智能指针安全的增减操作？

  13. 讲讲 C++ 中构造函数、析构函数、拷贝构造函数。

  14. new / malloc，如何 free 一个 class，过程什么样的？

  15. delete[] 一个数组，delete[] 如何知道要释放的内存多大？额外存储内存大小，逆序删除。

  16. 指针和引用的区别。数组的内容可以是引用吗？不可以。

  17. 多维数组的变量名的含义？

  18. 【右值引用】C++ 11 的右值引用，引用折叠，移动语义，完美转发。

  19. 宽窄字节怎么转换？

  20. C++ 有几种类型转换？

  21. 【面向对象】说说 C++ 的语言特点。说说面向对象的特性。封装、继承、多态。

  22. 讲讲重载、覆盖。（覆盖发生在继承过程）

  23. 讲讲运算符重载。不可以重载的运算符：sizeof :: . ? : .*

STL 库

  1. STL 库包含哪些内容？容器、迭代器、空间配置器、配接器、算法、仿函数。
  2. 看过 STL 库源码吗？印象最深的是什么？
  3. STL 库常用的容器有哪些？vector、list、map、unordered_map、set、queue、deque 等。
  4. vector 如何排序？vector 的扩容机制？vector 的迭代器实现机制？什么时候迭代器失效？
  5. list 和 vector 的区别。其实，在现代计算机上，list 的顺序访问和 vector 的任意访问，在计算速度上差异已经可以忽略不计了，但在实际使用中，还是会发现 list 比 vector 的速度慢了好几个数量级，为什么？链表 list 内存不连续，虚拟内存页访问的调度问题。（关联到操作系统 LRU 算法）
  6. map 和 unordered_map 的区别，插入、查找的复杂度。（关联数据结构的红黑树）
  7. unordered_map 扩容机制、阈值。
  8. set 和 hash_set 的区别。
  9. deque 的 insert 函数的使用

其它问题

  1. 多线程编程经验

  2. extern C 是什么，有什么作用？

  3. socket 网络编程了解吗？

  4. 了解 Python 吗？讲讲 Python 的内存回收机制。

  5. 了解什么脚本语言吗，比如 Lua？

  6. 调试命令（step into 等）

  7. 编译过程

    预处理（.c .h —-> 文本文件.i）替换 #include、宏定义等为真正的内容

    编译（.i —-> 汇编代码 .s）

    汇编（.s —-> 目标文件 .o）生成机器码

    链接（.o .so —-> 可执行文件）链接多个目标文件和库文件（.so）成最终可执行文件

数据结构

  1. 【查找】有哪些可以用于查找的数据结构？讲讲二叉查找树？查找的时间复杂度是多少？查找平均 $log_2N$，最坏 N，插入、删除和查找差不多。讲讲二分查？时间复杂度？
  2. 【排序】排序算法：归并排序、堆排序（堆的构建过程？为什么堆可以保证有序？有哪些操作保证了这个目标？）、桶排序、快排（如何避免倒序？如果有很多相同数字怎么办？）、插入排序（第 n 趟什么样子？）、冒泡排序，这些排序算法的时间复杂度，空间复杂度？
  3. 二叉树的遍历方式有哪些？前序、中序、后序、层序。
  4. 二叉平衡树、二叉查找树、红黑树的相同点和不同点是什么？从树高、查找、删除、插入复杂度几方面对比一下。
  5. map 底层怎么实现的？为什么用红黑树不用平衡二叉树？
  6. 【哈希表】设计一个哈希函数，要考虑哪些点？哈希表的实现原理？如何解决哈希冲突？开放定址法，拉链法。哪种方法更好？
  7. 和顺序比，链表存储有什么优缺点？
  8. 优先级队列，大小顶堆怎么实现的？找最大的 10 个数。

数据库

  1. SQL 删除最大 ID 的记录

操作系统

  1. 进程和线程有什么区别？最小的时间调度单位是什么？最小的资源分配单位是什么？进程间如何通信？代码怎么写？单线程、多线程通信方式？进程和线程的同步和通信方式？死锁的四个条件？我们通常写的并行程序是多线程还是多进程？多线程程序编写过程中最大的问题是什么？（共享资源的读写）如何避免这种问题？（加锁）如果发生死锁怎么办？从代码层面上说说。

    死锁必要条件：互斥、请求与保持、不可剥夺、循环等待

  3. 线程 sleep 和 wait 的区别

笔试编程题

  1. 从一个集合中找出最大的 K 个数。除优先级队列，可以用排序数组吗？在这个问题上，排序数组和优先级队列有什么区别？
  2. n 个鸡场，初始每个有 a[i] 只鸡，每天增长 k 只，并且每天最多的那个鸡场，卖一半鸡（向下取整），问 m 天后共几只鸡。
  3. n 个元素的数组，在它的连续子串（连续就是挨着的）中，随机取一个，然后取它的最大值，求这样取出的最大值的期望。（例如，[1、2、3]，连续子串有 [1]、[2]、[3]、[1, 2]、[2, 3]、[1, 2, 3]，[1, 2, 3] 的最大值就是 3，期望应该是 2.3333）
  4. n 个椅子抽取 m 个不相邻的，有多少种方法？
  5. 卖股票。小明买了的 n 支股票，行情不好要卖了，已知每支股票卖出会亏损的钱是 ai / 天，即如果第 k 天卖出则亏损 k * ai，小明一天最多允许卖 m 支股票，问如果打算卖出 q 支股票，最终亏损多少钱？
  6. n 个乘客，都有起点和终点，乘客订票顺序不一样，订票时，看到空座位，就可以选一个，求最少座位数，不论什么订票顺序，都够坐。
  7. 寻找二叉树两个节点的最近公共 parent。
  8. 有一个集合里面有很多元素 { a, b, c, d, … }，已知一个函数，可以返回集合中两两元素的直接连通关系（是否直接相连），比如：ab 相连，返回 true，bc 相连，返回 false，ac 不直接相连，返回 false，现在让你实现一个函数，使得所有相连的元素（包括直接相连和间接相连）放在同一个集合。
  9. 10 进制转 16 进制。
  10. 走台阶方法数（一次一阶 / 两阶）。
  11. 10000个人背上依次贴着从1到10000的编号，他们从小到大依次报数，偶数出列，一圈后，从剩下的5000人再次从号码小的开始报数，偶数出列，直到没人出列为止。问最后一次出列的人，背上的编号是多少？
  12. 密铺问题。
  13. 数字字符串切分，每段均可被 m 整除，有多少种切法？
  14. 聚集活动的病毒感染。
  15. 封城。
  16. 地图上有激光的，八方向的寻路问题，路径上激光不可连续。
  17. 乱序数组重排，操作只可以拿出一个数，放到最左端或者最右端。
  18. 一个 0/1 字符串，可以翻转一段连续字符，则 01 交替子串最大长度（不一定连续）？
  19. 40 米大刀。
  20. 对一堆集合的合并等操作。
  21. 一个排列到它的错排的最小加权距离。
  22. 推箱子。

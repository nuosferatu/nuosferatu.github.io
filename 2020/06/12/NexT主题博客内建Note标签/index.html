具体步骤：

很简单，在主题配置文件 _config.yml 中，note 下的 style 改为 simple、modern 或 flat 样式即可（当前这个是 modern），icons 可选 true 或 false 来开启或关闭左侧的小图标（当然也可以选择 true，然后在文章中对特定某个 Note 禁用 icon），light_bg_offset 是 modern 或 flat 样式的背景色，-12 颜色最深，12 颜色最浅（无色），当前是 7。

条件渲染

在 element 中，v-if="expression" 可以条件地控制 element 是否显示（实际上是控制 DOM 树是否生成该 element 节点），配合使用的还有 v-else 和 v-else-if，Vue 会根据条件不同来决定是否渲染 element 或者渲染哪个 element。

类似功能的还有 v-show="expression"，它和 v-if 的区别在于 DOM 始终是生成的，只是会根据条件决定是否 display。
